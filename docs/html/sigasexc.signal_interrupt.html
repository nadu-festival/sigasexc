classmethod subclass(sig: Union[signal.Signals, int], alias: Optional[str] = None) → Type[sigasexc.signal_interrupt.SignalInterrupt][source]

Parameters

  • sig (signal.Signals or int) – Signal number or signal.Signals member.

  • alies (str, optional, default None.) – The name of alies. You can access like SignalInterrupt.XXX if alies is “XXX”. If alies is None, alies sets to sig.name when sig is member of signal.Signals, alies sets to f”SIG{sig:02d}” when sig is integer.
